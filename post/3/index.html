触发器

触发器，就是制定一个规则，在我们做增删改操作的时候，只要满足该规则，自动触发，无需调用。
语句级触发器：不包含有for each row的触发器。
行级触发器：包含有for each row的就是行级触发器。
加for each row是为了使用:old或者:new对象或者一行记录。
